**This feature is only available in the Insiders Edition.**

:::

Karrio organizations allow you to manage teams or workspaces for a group of users. In a multi-tenant
context, each tenant, merchant or department can be represented as an organization.

The Management API allows you to create, update and delete organizations, teams and permissions programmatically.
